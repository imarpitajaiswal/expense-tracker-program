# Expense-Tracker-Program or Personal-Finance-Tracker

## 🎯 The Why
A lightweight, console-based personal finance management tool designed to help users track their daily spending with simplicity and efficiency. This project demonstrates foundational knowledge of Python programming, including data structure management and procedural logic.

## 🚀 Key Features
* **Expense Ingestion**: Easily record date, category, description, and amount for each transaction.
* **Transaction History**: Displays a clean, itemized list of all recorded expenses.
* **Financial Summary**: Automatically calculates and displays the total expenditure to provide immediate financial insights.
* **Robust Input Handling**: Designed with user-experience in mind to handle invalid menu selections gracefully.
